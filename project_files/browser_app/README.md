# Browser-only XLSX calculator

Статическая версия калькулятора: XLSX обрабатываются прямо в браузере, без backend и без загрузки файлов на сервер.

## Локальный запуск

```bash
pnpm install
pnpm dev
```

Открыть:

```text
http://127.0.0.1:8002
```

## Сборка для поддомена

```bash
pnpm build
```

Папку `dist/` можно раздать как обычную статику через nginx/CDN.

## Поведение

- Можно перетащить один или несколько `.xlsx` файлов.
- Все файлы конвертируются по очереди в браузере.
- Актуальные расчетные колонки: `T:W`.
- Колонки `X:AA` добавляются только как заголовки и не заполняются формулами.
- Формулы записываются в XLSX без кэшированных значений.
- Excel пересчитает формулы при открытии файла.
- Входные и выходные файлы не передаются на сервер.

## Проверка трансформера
