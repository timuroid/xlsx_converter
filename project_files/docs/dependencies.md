# Выявленные зависимости в примере `100019128.XLSX`

## Входная часть

SAP-выгрузка занимает колонки `A:S`.

Ключевые входные колонки:

- `O` - `Дата платежа`;
- `P` - `Дата выравнивания`;
- `Q` - `Вид документа`;
- `R` - `Сумма в ВВ`.

Расчет применяется только к строкам, где `Q = DR`.

## Расчетные колонки актуального MVP

Актуальный browser-only MVP добавляет расчетные колонки `T:AA`.

```text
T  Дней просрочки по КК (календарные)
U  % КК
V  Сумма КК
W  Сумма закрытия
```

В `AB1` хранится дата расчета. Она используется только для расчета `T`, если `P` пустая.

## Базовые правила

Для строк `DR`:

```text
если P заполнена:
  T = MAX(0, P - O)
иначе:
  T = MAX(0, $AB$1 - O)

U = 0.001
V = U * T * R
W = V + R
```

## Перерасчет X:AA

В выходной файл добавляются колонки `X:AA`:

```text
X  К доплате после пересчета платежа:
Y  к доплате в том числе по основному долгу
Z  к доплате в том числе по КК
AA Количество дней просрочки долга после пересчета
```

Формулы в `X:AA` ставятся только для строк `DR`, где `P` заполнена и `T > 0`:

```text
условие: Q = DR, P заполнена, T > 0
AA = MAX(0, $AB$1 - P)
Y  = V
Z  = AA * Y * U
X  = Y + Z
```
