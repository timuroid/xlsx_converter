# Change History

## 2026-07-07

- Создан проект для MVP XLSX-трансформера SAP-выгрузок УС.
- Добавлена базовая структура проекта и тестовый источник `100019128.XLSX`.
- Добавлен Python MVP на `openpyxl` с конфигом правил `rules.toml`.
- Добавлена проверка преобразования на реальном XLSX-примере.
- Добавлено легкое Flask web-приложение для загрузки и конвертации XLSX без постоянного хранения файлов.
- Добавлены Dockerfile и инструкция по развертыванию за reverse proxy на поддомене.
- Исправлена обработка чистой SAP-выгрузки без существующих Excel calculation properties.
- Добавлен тестовый источник `тестовая табличка_0.xlsx`.
- Добавлен browser-only POC на Vite/TypeScript/ExcelJS с drag-and-drop и пакетной конвертацией нескольких XLSX.
- Добавлена инструкция по статическому развертыванию browser-only версии на поддомене.
- Browser-only логика сужена до подтвержденных колонок `T:W`; `X:AA` перенесены в отложенные до уточнения правил.
- Browser-only интерфейс приведен к оранжевой стилистике презентации УС и заменен нативный выбор даты на стилизованный календарь.
- В browser-only интерфейс добавлена кнопка `Скачать все` для ZIP-архива нескольких готовых файлов, уменьшен календарь и исправлен выбор XLSX через клик по зоне загрузки.
- Проект подготовлен как локальный git-репозиторий: добавлен корневой `.gitignore`, уточнены `README.md` и `AGENTS.md` перед обсуждением GitLab-публикации с DevOps.
- Состав Git-репозитория сужен до browser-only приложения и проектной документации; локальные источники, исследования, Python fallback и результаты выгрузок исключены из отслеживания.
- Добавлены Dockerfile, docker-compose и nginx-конфиг для browser-only приложения; фон интерфейса дополнен декоративными линиями в стилистике презентации УС.
